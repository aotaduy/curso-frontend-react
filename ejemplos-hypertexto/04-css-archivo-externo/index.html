CSS en Archivo Externo

Este ejemplo demuestra la mejor práctica para aplicar estilos CSS: en un archivo externo separado del HTML.

Ventajas

  • Separación de contenido y presentación
  • Reutilización de estilos en múltiples páginas
  • Fácil de mantener y actualizar
  • El navegador cachea el archivo CSS (mejor rendimiento)
  • HTML más limpio y legible
