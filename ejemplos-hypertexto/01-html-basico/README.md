# HTML Básico

**Concepto:** Estructura mínima de un documento HTML

## Archivo
- `index.html` - Documento HTML simple con estructura básica

## Conceptos Demostrados
- `<!DOCTYPE html>` - Declaración de documento
- `<html>` - Elemento raíz
- `<head>` - Metadatos e información del documento
- `<meta charset>` - Codificación UTF-8
- `<title>` - Título de la página
- `<body>` - Contenido visible
- Etiquetas básicas: `<h1>`, `<p>`, `<a>`, `<img>`, `<ul>`, `<li>`

## Cómo Abrir
1. Doble-click en `index.html` o
2. Abre en VS Code y usa "Open with Live Server" o
3. Abre manualmente en el navegador

## Resultado Esperado
Una página simple con encabezados, párrafos, enlaces, listas e imágenes.
