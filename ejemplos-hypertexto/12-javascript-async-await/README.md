# JavaScript Async Await

**Concepto:** escribir codigo asincrono con estilo secuencial.

## Archivo
- `index.html` - Flujo asincrono con `await` y manejo de errores

## Conceptos demostrados
- Declaracion de funciones `async`
- Uso de `await` sobre promesas
- Manejo de errores con `try/catch`

## Como probar
1. Abre `index.html`
2. Presiona "Ejecutar flujo async"
3. Revisa el orden de ejecucion en pantalla y consola

## Version Node.js
- Archivo equivalente: `node-example.js`
- Descargar archivo JS: [node-example.js](./node-example.js)

### Como correr en Node.js
1. Abre una terminal en esta carpeta.
2. Ejecuta `node node-example.js`.
