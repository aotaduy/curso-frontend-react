# JavaScript Variables y Funciones

**Concepto:** Fundamentos del lenguaje JavaScript

## Archivo
- `index.html` - Ejemplos con código en `<script>` al final y en consola

## Conceptos Demostrados
- Variables: `let`, `const`, `var`
- Tipos de datos: String, Number, Boolean, Array, Object
- Tipado dinámico (cambiar tipo en tiempo de ejecución)
- Funciones declarativas
- Arrow functions `() => {}`
- Operadores matemáticos: `+`, `-`, `*`, `/`, `%`, `**`
- Comparadores: `===`, `==`, `>`, `<`, `>=`, `<=`
- Lógica booleana: `&&`, `||`, `!`
- Condicionales: `if/else`, `switch`
- Arrays y métodos: `push`, `pop`, `map`, `filter`, `reduce`
- Objetos y propiedades
- Closures (funciones anidadas)
- Destructuring (desempaquetado)
- Template literals (strings mejorados)

## Cómo Probar
1. Abre `index.html` en navegador
2. Abre DevTools (F12)
3. Ve a Console (pestaña Console)
4. Verás los logs ejecutables automáticamente

## Código Ejecutable
El archivo tiene ejemplos que se ejecutan automáticamente y muestran results en la consola.

## Comparación con Otros Lenguajes
- Similar a **Python** en tipado dinámico
- Diferente de **C** en no tener tipos fijos
- Las funciones son **first-class** (pueden pasarse como valores)

## Resultado Esperado
- Ejemplos visibles en página
- Muchos console.logs en la consola (F12)
- Comprensión de variables, funciones y tipos
