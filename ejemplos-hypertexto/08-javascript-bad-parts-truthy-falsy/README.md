# JavaScript Bad Parts: Truthy y Falsy

**Concepto:** coercion implicita y valores que pueden confundir en condicionales.

## Archivo
- `index.html` - Tabla y logs de valores truthy/falsy

## Conceptos demostrados
- Valores falsy: `false`, `0`, `-0`, `0n`, `""`, `null`, `undefined`, `NaN`
- Valores truthy comunes: `"0"`, `[]`, `{}`, numeros no cero
- Diferencia entre `==` y `===`

## Como probar
1. Abre `index.html` en navegador
2. Abre DevTools (F12)
3. Revisa tabla y logs en consola

## Version Node.js
- Archivo equivalente: `node-example.js`
- Descargar archivo JS: [node-example.js](./node-example.js)

### Como correr en Node.js
1. Abre una terminal en esta carpeta.
2. Ejecuta `node node-example.js`.
