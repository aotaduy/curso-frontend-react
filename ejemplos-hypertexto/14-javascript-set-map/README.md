# JavaScript Set y Map

**Concepto:** estructuras de datos para colecciones sin duplicados y diccionarios avanzados.

## Archivo
- `index.html` - Demo de uso de `Set` y `Map`

## Conceptos demostrados
- `Set` para eliminar duplicados
- `Map` para pares clave/valor
- Metodos: `.set()`, `.get()`, `.has()`, `.size`
- Iteracion con `for...of`

## Como probar
1. Abre `index.html`
2. Presiona "Ejecutar demo"
3. Revisa salida en pantalla y consola

## Version Node.js
- Archivo equivalente: `node-example.js`
- Descargar archivo JS: [node-example.js](./node-example.js)

### Como correr en Node.js
1. Abre una terminal en esta carpeta.
2. Ejecuta `node node-example.js`.
