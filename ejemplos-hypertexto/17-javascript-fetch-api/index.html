JavaScript: Fetch API

La Fetch API permite hacer peticiones HTTP asincrónicas al servidor (o APIs externas) sin recargar la página. Usaremos JSONPlaceholder, un servicio gratuito que proporciona datos ficticios para pruebas.

💡 Consejos:

  • Abre el Network tab (F12 → Network) para ver las peticiones HTTP
  • Usa la Console (F12 → Console) para ver logs y errores
  • JSONPlaceholder ralentiza las respuestas con ?_delay=1000 para ver loading states
  • Las peticiones POST no se guardan permanentemente en JSONPlaceholder
