CSS Flexbox (Layout 1D)

Flexbox es una herramienta poderosa para crear layouts responsivos. A diferencia de Grid, Flexbox trabaja en una dimensión (fila o columna).

1. Flex Container Básico (Fila)

display: flex + gap: 20px

Tarjeta 1

Contenido responsivo. Los items se adaptan al tamaño de pantalla.

Tarjeta 2

Usa flex: 1 para crecer equitativamente.

Tarjeta 3

Se hacen una columna en pantallas pequeñas.
