# JavaScript Closures

**Concepto:** una funcion interna recuerda el scope donde fue creada.

## Archivo
- `index.html` - Dos contadores independientes creados con closures

## Conceptos demostrados
- Estado privado con variables locales
- Factory function (`createCounter`)
- Multiples instancias con estado separado

## Como probar
1. Abre `index.html`
2. Presiona los botones de contador A y B
3. Observa que cada uno mantiene su propio valor

## Version Node.js
- Archivo equivalente: `node-example.js`
- Descargar archivo JS: [node-example.js](./node-example.js)

### Como correr en Node.js
1. Abre una terminal en esta carpeta.
2. Ejecuta `node node-example.js`.
