CSS Grid (Layout 2D)

Grid es perfecto para layouts en 2 dimensiones. Controlas filas Y columnas simultáneamente.

1. Grid Básico - 3 Columnas

grid-template-columns: repeat(3, 1fr)

Producto 1

Layout responsivo con 3 columnas iguales.

Producto 2

En pantallas pequeñas se convierte en 1 columna.

Producto 3

Usa gap para el espaciado entre items.

Producto 4

Todos los items tienen el mismo tamaño.

Producto 5

¡Muy útil para tiendas y galerías!

Producto 6

Grid es muy flexible.

4. Grid Responsivo Automático

grid-template-columns: repeat(auto-fit, minmax(250px, 1fr))

Se adapta automáticamente: 4 columnas en pantalla grande, 2 en mediana, 1 en móvil.
