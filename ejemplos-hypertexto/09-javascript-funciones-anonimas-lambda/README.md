# JavaScript Funciones Anonimas y Lambda

**Concepto:** funciones como valores y sintaxis arrow.

## Archivo
- `index.html` - Comparacion de callbacks anonimos y lambdas

## Conceptos demostrados
- Funcion anonima: `function(x) { ... }`
- Funcion lambda: `(x) => ...`
- Uso de callbacks con `map`
- Return implicito en lambdas

## Como probar
1. Abre `index.html`
2. Presiona "Ejecutar ejemplos"
3. Revisa consola y salida en pantalla

## Version Node.js
- Archivo equivalente: `node-example.js`
- Descargar archivo JS: [node-example.js](./node-example.js)

### Como correr en Node.js
1. Abre una terminal en esta carpeta.
2. Ejecuta `node node-example.js`.
