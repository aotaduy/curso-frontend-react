# Desarrollo Web - Apunte del Curso

## Introducción

Este apunte cubre los fundamentos de tecnologías web necesarios para desarrollar aplicaciones frontend y backend modernas. Está diseñado para estudiantes de Ingeniería en Sistemas de la UTN con conocimientos básicos de programación (Python, Smalltalk, C) pero sin experiencia previa en JavaScript.

El material está organizado en tres documentos principales que se deben leer en orden:

### 📚 Documentos del Curso

1. **[WWW.md](WWW.md)** - Fundamentos de la World Wide Web
   - Historia y tecnologías
   - Protocolos: TCP/IP, DNS, HTTP, URLs
   - Servidores web: Apache, Nginx, Node.js, PHP, Java
   - Navegadores: renderizado, event loop, APIs

2. **[Hypertexto.md](Hypertexto.md)** - Tecnologías Frontend Base
   - HTML: estructura de documentos web
   - CSS: estilos y diseño visual
   - JavaScript: comportamiento e interactividad
   - Frameworks: introducción a Angular y React

3. **[React.md](React.md)** - Librería React para UI Modernas
   - JSX y componentes
   - Hooks (useState, useEffect, useReducer)
   - Context para estado global
   - Manejo de estado asíncrono y llamadas a APIs
   - Client-side routing
