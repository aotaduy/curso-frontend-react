# Glosario de Siglas y Jerga Web

Este glosario evita redundancia: incluye solo terminos que aparecen en `WWW.md`, `Hypertexto.md` y `React.md` pero que no tienen una seccion propia de definicion en esos archivos.

## Terminos

### UI
Definicion corta: User Interface. Es la interfaz visual con la que interactua el usuario (botones, formularios, vistas, feedback).

Se usa en: [React.md](React.md)

MDN/Docs: https://developer.mozilla.org/en-US/docs/Glossary/UI
Wikipedia/Referencia: https://es.wikipedia.org/wiki/Interfaz_de_usuario

### MVC
Definicion corta: Patron Model-View-Controller que separa datos/logica (Model), interfaz (View) y coordinacion de acciones (Controller).

Se usa en: [React.md](React.md)

MDN/Docs: https://developer.mozilla.org/en-US/docs/Glossary/MVC
Wikipedia/Referencia: https://es.wikipedia.org/wiki/Modelo%E2%80%93vista%E2%80%93controlador

### Diffing
Definicion corta: Proceso de comparar dos estructuras (por ejemplo, dos versiones del DOM virtual) para detectar y aplicar solo cambios minimos.

Se usa en: [React.md](React.md)

MDN/Docs: https://react.dev/learn/render-and-commit
Wikipedia/Referencia: https://en.wikipedia.org/wiki/Diff

### Routing
Definicion corta: Mapeo entre URL/rutas y componentes o vistas de una aplicacion.

Se usa en: [React.md](React.md)

MDN/Docs: https://reactrouter.com/en/main/start/overview
Wikipedia/Referencia: https://en.wikipedia.org/wiki/Routing

### CORS
Definicion corta: Politica de seguridad del navegador que controla solicitudes HTTP entre origenes diferentes.

Se usa en: [WWW.md](WWW.md)

MDN/Docs: https://developer.mozilla.org/en-US/docs/Web/HTTP/CORS
Wikipedia/Referencia: https://es.wikipedia.org/wiki/Cross-origin_resource_sharing

### JSON
Definicion corta: Formato de intercambio de datos basado en texto, comun en APIs HTTP y frontend React.

Se usa en: [WWW.md](WWW.md), [React.md](React.md)

MDN/Docs: https://developer.mozilla.org/en-US/docs/Glossary/JSON
Wikipedia/Referencia: https://es.wikipedia.org/wiki/JSON

### Prop Drilling
Definicion corta: Pasar props por varios componentes intermedios que no usan esos datos, solo para llegar a un componente hijo profundo.

Se usa en: [React.md](React.md)

MDN/Docs: https://react.dev/learn/passing-data-deeply-with-context
Wikipedia/Referencia: https://en.wikipedia.org/wiki/Prop_drilling

### CRUD
Definicion corta: Operaciones basicas sobre datos: Create, Read, Update y Delete.

Se usa en: [React.md](React.md), [WWW.md](WWW.md)

MDN/Docs: https://developer.mozilla.org/en-US/docs/Glossary/CRUD
Wikipedia/Referencia: https://es.wikipedia.org/wiki/CRUD
